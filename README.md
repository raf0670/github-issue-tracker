1️⃣ var is function-scoped and hoists. let is block-scoped and reassignable. const is block-scoped and cannot be reassigned.

2️⃣ The spread operator (...) expands an iterable (like an array or object) into individual elements or properties.

3️⃣ map() creates a new array by transforming every element. filter() creates a new array with elements that pass a test. forEach() simply executes a function for each element without returning anything.

4️⃣ An arrow function is a concise way to write functions using the => syntax that does not have its own this binding.

5️⃣ Template literals are strings in backticks (`) and we can write multi-line text and variables using ${expression} syntax.
